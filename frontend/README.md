# Frontend (Open-source Agent Chat UI adaptation)

This frontend is built with Next.js and adapted from the open-source chat UI kit:

- `@chatscope/chat-ui-kit-react` (MIT)

It renders:

- ChatPanel
- WorkflowPanel
- NodeDetailPanel
- ToolCallPanel
- ArtifactPanel

Run:

```powershell
npm install
$env:NEXT_PUBLIC_BACKEND_URL="http://localhost:8000"
npm run dev
```
